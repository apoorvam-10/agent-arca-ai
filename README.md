
# 🧠 Agent ARCA - AI Research Assistant

## 🚀 Overview
Agent ARCA is a multi-agent AI system that:
- Ingests web pages and PDFs
- Extracts and processes information
- Generates AI-powered summaries using Gemini
- Provides evaluation metrics

---

## ⚙️ Tech Stack
- Python
- Streamlit
- Google Gemini API
- BeautifulSoup
- PyMuPDF

---

## 🧠 Features
- Multi-source ingestion (URL + PDF)
- AI-powered summarization
- Clean web UI
- Modular pipeline design

---

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
